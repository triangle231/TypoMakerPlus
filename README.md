# TypoMaker+

TypoMaker+는 텍스트, 도형, 이미지를 조합해 한글 타이포그래피 애니메이션을 제작할 수 있는 웹 기반 편집기입니다. 타임라인과 프리뷰 환경을 제공하여 영상 같은 모션을 손쉽게 연출할 수 있습니다.

## 핵심 기능

- 수십 종의 한글 폰트와 스타일 옵션을 활용한 텍스트 애니메이션
- 도형, 이미지, 배경 채우기 클립을 포함한 레이어 기반 구성
- 위치·크기·회전·투명도를 제어할 수 있는 키프레임 애니메이션
- 엔트리(Entry)용 `.ent` 패키지와 JSON 프로젝트 내보내기/불러오기

## 빠른 시작

1. 저장소를 다운로드하거나 클론합니다.
2. 어떤 정적 서버든 루트(`index.html`)를 서비스하도록 실행하거나, 파일을 브라우저로 직접 열어도 됩니다.
3. 브라우저에서 편집 화면이 열리면 타임라인과 패널을 이용해 클립을 추가하고 설정을 조정합니다.

권장 브라우저: 최신 Chrome, Edge, Firefox, Safari.

## 기본 사용법

- **에셋 패널**에서 텍스트, 도형, 이미지, 배경 채우기를 추가합니다.  
- **속성 패널**에서 선택한 클립의 글꼴, 색상, 배경, 애니메이션 수치를 설정합니다.  
- **타임라인**에서 클립 길이를 조정하고, 키프레임 그래프에서 easing과 시작/끝 값을 입력하여 동작을 정의합니다.  
- **상단 컨트롤**을 통해 재생·정지, 확대/축소, 트랙 추가/복제를 수행할 수 있습니다.  
- **내보내기** 버튼으로 JSON 또는 `.ent` 파일을 생성하고, **불러오기** 버튼으로 저장된 프로젝트를 다시 열 수 있습니다.

## 편집 가이드

### 패널 구성

왼쪽 패널은 `에셋`, `레이어`, `속성` 탭으로 나뉩니다. 에셋 탭에서는 미리 준비된 텍스트/도형 프리셋과 이미지 자산을 추가할 수 있고, 레이어 탭은 현재 타임라인에 존재하는 모든 클립을 트랙 순서대로 보여주며 드래그로 선택할 수 있습니다. 속성 탭은 선택한 클립의 상세 설정을 제공하며, 아무 것도 선택하지 않은 경우 프로젝트 배경색, 배경음악, 볼륨 등을 제어할 수 있습니다.

### 클립 추가와 편집

에셋을 클릭하면 현재 선택된 트랙과 재생 헤드 위치를 기준으로 새 클립이 생성됩니다. 생성된 클립은 타임라인에서 드래그해 위치와 길이를 조정하며, 속성 탭에서 텍스트 내용, 폰트, 색상, 도형 크기, 이미지 이름 등 필수 정보를 수정합니다. 텍스트 클립은 굵게/기울임/밑줄 토글과 배경 색상, 투명도 옵션을 함께 제공합니다.

### 타이밍과 위치 제어

- `시작 시간`/`종료 시간` 입력란은 초 단위로 동작하며, 소수점 첫째 자리까지 입력하면 0.1초 단위로 정밀 수정할 수 있습니다.  
- 타임라인에서 클립을 드래그하면 시작 시간이 자동으로 갱신되고, 우측 손잡이를 움직이면 종료 시간이 재계산됩니다.  
- 키보드 `Shift`를 누른 채 드래그하면 0.1초 단위 스냅이 적용되어 미세 조정이 쉬워집니다.  
- `Alt`+드래그는 클립을 복제한 뒤 이동하며, 복제된 클립은 동일한 속성을 유지합니다.

### 애니메이션 설정

`애니메이션` 섹션에는 X·Y 위치, 크기(%), 회전(도), 투명도(%)에 대한 그래프가 표시됩니다. 각 그래프에서 시작값과 끝값을 입력하거나 입력 필드 좌우의 화살표 버튼을 사용해 수치를 빠르게 수정할 수 있습니다. easing 드롭다운은 `Linear`, `Ease In`, `Ease Out`, `Ease In-Out` 네 가지를 지원하며, 그래프 캔버스를 클릭하면 시각적 곡선이 즉시 재계산됩니다.

- X/Y 위치 값은 캔버스 중앙을 기준으로 하며, 픽셀 단위 오프셋을 의미합니다.  
- 크기(Scale)는 100이 원본 크기이며, 200은 2배, 50은 절반 크기를 뜻합니다.  
- 회전은 도(degree) 단위이며, 양수는 시계방향, 음수는 반시계 방향 회전을 나타냅니다.  
- 투명도는 0이 완전 투명, 100이 완전 불투명입니다.  
- 그래프 하단의 입력란에 `초기값 → 종료값`을 입력하면 자동으로 키프레임이 설정되며, 재생 헤드 위치에서 값을 수정하면 해당 시점에 새 키프레임이 추가됩니다.

### 키프레임과 타임라인 조작

타임라인의 클립은 양 끝 손잡이를 드래그해 길이를 바꾸고, 트랙 간 드래그 앤 드롭으로 레이어 순서를 재배치합니다. 재생 헤드를 기준으로 `Ctrl + C`, `Ctrl + V`로 클립 복사·붙여넣기가 가능하며, 컨텍스트 메뉴(우클릭)에서 복제, 숨기기, 삭제 등을 선택할 수 있습니다. 다중 선택은 `Shift`+클릭 또는 드래그 박스(빈 공간 드래그)로 수행하며, 선택된 클립에 대해 일괄 삭제 및 이동이 가능합니다.

- 클립 이동 시 상단 자 모양 눈금에 스냅되며, `Shift`를 누르면 세밀 이동으로 전환됩니다.  
- `Ctrl`을 누른 채 손잡이를 드래그하면 클립 길이가 늘어난 부분만큼 종료 시간이 확장되고, 시작 시간은 고정됩니다.  
- 숨겨진 클립은 타임라인에서 약간 어둡게 표시되며, 레이어 패널 또는 컨텍스트 메뉴에서 다시 표시할 수 있습니다.

### 카메라 모드

타임라인 우측 상단 모드 스위치를 `카메라`로 바꾸면 카메라 트랙이 나타납니다. 빈 공간에서 우클릭하면 Linear/EaseIn/EaseOut/EaseInOut 키프레임을 추가할 수 있고, 추가된 키프레임을 드래그해 시간과 위치를 조정합니다. 속성 패널의 카메라 키프레임 데이터는 `.ent` 내보내기 시 함께 포함됩니다.

- 각 카메라 키프레임은 X/Y 위치, 확대 비율(Scale), 방향(Rotation)을 가진다.  
- 키프레임을 선택하면 속성 패널에 해당 값이 표시되며, 숫자 입력으로 미세 조정 가능하다.  
- `Shift`를 누른 채 키프레임을 드래그하면 눈금에 스냅되어 정확한 타이밍 세팅이 편하다.  
- 카메라 트랙의 컨텍스트 메뉴를 이용하면 키프레임 복사·붙여넣기가 가능하고, 복제 시 easing과 값이 함께 복사된다.

### 미리보기와 변형

미리보기 영역의 오브젝트는 드래그로 위치 이동이 가능하며, 클립을 선택하면 변형 핸들이 표시되어 크기와 회전을 조정할 수 있습니다. 변형 결과는 해당 클립의 애니메이션 시작값에 실시간 반영되므로, 정확한 수치를 원할 경우 속성 패널에서 값 입력 후 미리보기로 미세 조정하면 효율적입니다.

- `Shift`+드래그를 사용하면 비율이 유지된 채 크기를 조정합니다.  
- `Alt`+드래그는 현재 위치를 기준으로 대칭 변형을 수행합니다.  
- 회전 핸들을 드래그할 때 `Shift`를 누르면 15도 단위로 스냅됩니다.  
- 변형 후 `Ctrl + Z`로 즉시 되돌릴 수 있으며, 속성 패널의 값을 직접 입력해도 변형 상태와 동기화됩니다.

### 배경음악과 볼륨

`배경음악` 버튼을 눌러 오디오 파일을 추가하면 전체 타임라인 재생과 동기화됩니다. 볼륨 슬라이더는 0~100%를 지원하며, `.ent`로 내보낼 때도 퍼센트 값이 저장됩니다. 이미 선택된 음악이 있을 때 새 파일을 추가하면 기존 오디오 객체는 자동으로 갱신됩니다.

### 프로젝트 저장과 공유

`JSON 다운로드`는 프로젝트 편집 정보를 순수 JSON으로 저장해 두고 나중에 `프로젝트 불러오기`로 복원할 때 사용합니다. `ENT 파일로 내보내기`는 Entry 호환 패키지를 생성하며, 내보내기 과정에서 모든 텍스트/도형이 PNG로 렌더링되어 tarball 내부에 포함됩니다. 내보내기 진행률은 전체 자산 처리 단계에 맞춰 표시되고, 완료 후 자동으로 `project.ent` 파일이 내려받아집니다.

## 단축키

- `Space` 재생/일시정지  
- `Ctrl + Z / Ctrl + Y` 실행 취소·다시 실행  
- `Ctrl + C / Ctrl + V` 클립 복사·붙여넣기  
- `Delete` 선택된 클립 삭제
